# IP Reputation Checker

## 简介
这是一个用于检测 IP 地理位置并查询 IP 纯净度的网站，能够帮助用户快速了解当前 IP 的基本信息与风险情况。

## 功能
- 查询 IP 地理位置
- 检测是否为代理 IP
- 检测是否为数据中心 IP
- 提供 IP 风险评分
- 辅助判断 IP 信誉与纯净度

## Introduction
This is a web-based tool for detecting IP geolocation and checking IP cleanliness/reputation.  
It helps users quickly understand the basic information and potential risks associated with an IP address.

## Features
- IP geolocation lookup
- Proxy IP detection
- Datacenter IP detection
- IP risk scoring
- Reputation and cleanliness evaluation
